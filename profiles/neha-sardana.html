Pattern Matching for Java

Pattern Matching is already known to Java Developers. Over time, the Java language has expanded its pattern matching capabilities from just matching strings to matching object types. The introduction of Records and Sealed Classes in previous Java versions is just part of the story.

In this talk, we will discuss the need and usage of Pattern Matching. We will uncover this feature which is going to be most helpful to developers going forward.
